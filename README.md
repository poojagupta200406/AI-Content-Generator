AI-Content-Generator
A web app that generates emails, advertisements, captions, and product descriptions using the Groq API (Llama 3.3 70B).

What it does
You give it a topic, pick a content type and tone, set a word limit — it writes the content for you.
Content types: Email · Advertisement · Caption · Product Description
Tones: Formal · Casual · Professional · Friendly · Persuasive

Tech stack:
Python / Flask
Groq API — Llama 3.3 70B Versatile
HTML, CSS,
python-dotenv
